# Product Overview

**Su Rede Social** is a social media content creation and management platform for the brand "SU Controle" (a personal finance app).

## Core Purpose
AI-powered tool that helps create, schedule, and analyze social media content across platforms (Instagram, LinkedIn, TikTok, Twitter).

## Key Features
- **Brand Setup**: Onboarding flow that analyzes website/Instagram to extract brand identity (colors, tone, niche)
- **Content Creator**: Generate posts, images, videos, and carousels using Google Gemini AI
- **Calendar/Agenda**: Weekly content planning with AI-generated topic suggestions
- **Dashboard**: Real-time analytics showing post metrics, platform distribution, and activity trends
- **Templates**: Pre-designed visual templates (Educational, Quote, Minimal Dark) for quick content creation

## Target User
Social media managers and content creators for the SU Controle brand who need to produce consistent, on-brand content efficiently.
